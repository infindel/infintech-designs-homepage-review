## What this is

10 homepage design directions for [infintechdesigns.com](https://www.infintechdesigns.com/), presented in one reviewable gallery so the client can browse, filter, preview full-screen, and compare any two side-by-side.

- **Structural (A):** 4 styles — Bento Grid · Asymmetric · Full-Screen Hero · Card-Based
- **Visual (B):** 4 styles — Dark Mode · Maximalism · Typographic · Corporate / Professional
- **Functional (C):** 2 styles — Landing Page · Portfolio

## How to review

1. Open the link above
2. Click **Start Reviewing** on the landing page
3. Use the **Filter** tabs to narrow by category
4. **Preview** any style full-screen, use arrow keys to cycle
5. Toggle **Compare Two** top-right to see any pair side-by-side

No login needed. Works on phone, tablet, or laptop.

- Shortlist top 3: ~20 minutes
